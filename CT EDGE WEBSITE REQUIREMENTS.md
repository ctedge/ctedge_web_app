**CT EDGE LTD – WEBSITE DEVELOPMENT REQUIREMENTS**

**1\. Project Objective**

Transform the current CT Edge Ltd website into a full real estate and investment platform that supports property listings, investor participation, customer dashboards, and online payments.

**2\. Current Website Summary**

Current website: https://ctedgeltd.com

Existing Features:

• Home page

• Basic company introduction

• Contact information

• WhatsApp contact

Limitations:

• No property listings

• No housing listings

• No investment portal

• No customer accounts

• No payment system

• No project portfolio

• No inspection booking system

**3\. Main Public Website Pages**

The new website should include the following pages

• Home

• About

• Land

• Houses

• Construction Services

• Projects

• Investment Opportunities

• Blog / Insights

• Contact

**4\. Property Platform Features**

Land Listings:

• Estate name

• Location

• Plot size

• Price

• Title document

• Payment plan

• Estate features

• Photo gallery

• Map location

• Download brochure

• Book inspection button

House Listings:

• House type

• Description

• Price

• Payment plan

• Floor plan

• Image gallery

• Reserve property button

**5\. Inspection Booking System**

Users should be able to book site inspections through a form containing:

• Name

• Phone

• Email

• Property interested in

• Preferred inspection date

• Message

Admin should receive email notifications for inspection bookings.

**6\. Customer Dashboard System**

Customers should be able to create accounts and log in.

Dashboard features:

• View purchased property

• Track installment payments

• Payment history

• Outstanding balance

• Download receipts

• Upload payment proof

• Download allocation documents

**7\. Investor Portal**

Investors should be able to create accounts and manage investments.

Features:

• Investor login

• Investment opportunities listing

• Invest directly in projects

• ROI projections

• Investment duration

• Investment maturity dates

• Download investment agreements

Investor dashboard should display:

• Total investment

• Expected return

• Investment duration

• Payment updates

**8\. Payment System**

Website should integrate secure online payment systems.

Supported payments:

• Full payment

• InstallmEnt payments

Features:

• Secure payment gateway (Paystack or Flutterwave)

• Automatic email receipts

• Payment confirmation notifications

• Payment history tracking

**9\. Admin Management System**

Admin should be able to:

• Upload land and housing products

• Manage projects

• Manage customer accounts

• Track installment payments

• Manage investors

• Approve investments

• Generate financial reports

• Send notifications to customers and investors

**10\. Automation Features**

• Automatic payment receipts

• Installment payment reminders

• Investment maturity reminders

• Email notifications

• Inspection booking confirmations

**11\. Additional Features**

• WhatsApp chat integration

• Live chat support

• Property galleries and videos

• Google Maps integration

• Download brochure option

• Mobile responsive design

• SEO optimized structure
